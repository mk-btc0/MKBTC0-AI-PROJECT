## 🚀 About

**MKBTC0 AI Hub** is a web platform about Generative AI. Features: AI news (GNews API), 25+ AI tools, prompt guides, CMD console with commands (/help, /logs, /back), quiz, activity logs, model comparison (GPT-4/Claude/Gemini), and login system.

## 🛠️ Tech Stack

`HTML` • `CSS` • `JavaScript` • `LocalStorage` • `GNews API`

## 📂 Files

- `index.html` — home page
- `allAI.html` — AI directory
- `news.html` — AI news
- `console.html` — CMD terminal
- `logs.html` — activity logs
- `glossary.html` — AI dictionary
- `login.html` — authentication
