📖 PROMPT ENGINEERING GUIDE

✏️

Basic Structure

[ROLE] + [TASK] + [CONTEXT] + [FORMAT]

Example: "You are a senior developer. Write Python code for a REST API using FastAPI. Return only the code."

⚡

Advanced Techniques

  • Few-shot: "Here are examples: ... Now do the same"
  • Chain-of-Thought: "Let's think step by step"
  • Negative prompting: "Don't use complex terminology"
